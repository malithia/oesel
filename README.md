# OESEL: Ontology of English and Sinhala Emotion Lexicon

This repository contains the `.owl` file for OESEL, an ontology of emotion lexicon for English and Sinhala. 
The contents of this repository are provided as supplementary material for the paper “Things, not strings”: Laying the conceptual groundwork for an Ontology of English and Sinhala Emotion Lexicon.
